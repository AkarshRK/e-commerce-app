# React.js exommerce application with redux and redux-thunk

## Requirements:
- React.js
- NodeJS and npm

## Installation

Clone the above repo:
```sh
git clonehttps://github.com/AkarshRK/e-commerce-app.git
cd e-commerce-app/ecom
```

Install the react packages...

```sh
npm install
```

Run the code...

```sh
npm run 
```
